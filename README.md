# Risk Signals in Consumer Lending

This repository contains a GitHub Pages version of my CS 432 project journal.

The site is organized as one page per course module, following the eleven modules of the class and showing how the project developed from the introduction and data work through clustering, dimensionality reduction, regression, classification, support vector machines, ensemble methods, and the final write up.

## How to publish with GitHub Pages

1. Upload all files and folders in this directory to a public GitHub repository.
2. Go to Settings > Pages.
3. Under Build and deployment, choose Deploy from a branch.
4. Select the main branch and /root folder.
5. Save and wait for GitHub Pages to publish the site.

## Contents

- `index.html`: project homepage
- `modules/`: one HTML page for each course module
- `assets/`: selected visuals and project report
- `code/`: selected Python scripts from the project
